# 🛒 Retail Analytics Project

## 📌 Overview

This project focuses on analyzing retail sales data to generate meaningful business insights using an end-to-end data analytics pipeline.
It covers data cleaning, database management, analysis, and dashboard visualization.

---

## 🎯 Objective

To transform raw retail data into actionable insights that help in:

* Identifying top-performing products and customers
* Analyzing regional sales performance
* Understanding profit distribution across categories
* Supporting data-driven decision making

---

## 🛠️ Tools & Technologies Used

* **Excel** → Data Cleaning & Preprocessing
* **MySQL** → Database Creation & Querying
* **Python (Pandas + SQLAlchemy)** → Data Import & Automation
* **Power BI** → Interactive Dashboard & Visualization

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Excel)

* Removed inconsistencies and formatted data
* Prepared structured dataset for analysis

### 2️⃣ Data Storage (MySQL)

* Created database: `retail_project`
* Imported cleaned dataset into SQL table
* Performed analytical queries

### 3️⃣ Data Pipeline (Python)

* Used Pandas to load CSV
* Connected MySQL using SQLAlchemy
* Automated data upload process

### 4️⃣ Data Visualization (Power BI)

* Built interactive dashboard with KPIs:

  * Total Sales
  * Total Profit
  * Total Orders
  * Average Order Value (AOV)

---

## 📊 Key Insights

* 📍 **West region** generated the highest sales
* 💻 **Technology category** contributed maximum profit
* 🏆 Top products drive a significant portion of revenue
* 👥 A few customers contribute heavily to total sales
