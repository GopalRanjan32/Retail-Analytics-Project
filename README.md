# Retail Analytics Project

## Overview
This project analyzes retail sales data using Excel, SQL, Python, and Power BI.

## Tools Used
- Excel (Data Cleaning)
- MySQL (Data Storage & Queries)
- Python (Data Import)
- Power BI (Dashboard)

## Key Insights
- Total Sales & Profit
- Top Products
- Top Customers
- Region-wise Performance
